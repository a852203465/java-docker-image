## 3.项目分支说明

* Active branches:
    * `master` - Latest Java, currently version 17
* Inactive branches:
    * `java17` - Java 17
    * `java11` - Java 11
    * `java8` - Java 8

 - alpine-: 以`alpine:15`为基础镜像制作
 - debian-: 以`debian:11`为基础镜像制作
 - centos: 以`centos:7`为基础镜像制作
 - -jdk: 包含JDK完整环境
 - -jre: 仅JRE运行环境
